# Creating Your ERC-20 Token

### Effortless Token Deploying

<figure><img src="../.gitbook/assets/image.png" alt=""><figcaption><p>Sonic Factory Interface</p></figcaption></figure>

Creating your token is quick and straightforward. Simply click **Create Token** on the navigation bar, enter the necessary token details, and hit **Deploy**. With just a few steps, your token is ready to go, making the process efficient and hassle-free. No complex coding or setupâ€”just a smooth, user-friendly experience!

| Token Details | Input Type | Example  |
| ------------- | ---------- | -------- |
| Token Name    | String     | Ethereum |
| Token Symbol  | String     | ETH      |
| Token Supply  | Number     | 1000000  |
